About Eduread

Eduread - Education HTML5 Templateis a responsive clean and modern HTML5 Bootstrap Education Template for any School website, College website, Universities and other Educational Institutions. This template includes 4 different homepage with 25+ html pages carefully designed blocks, which are easy to edit and customize to fit your needs. Eduread is powered by the Bootstrap 3.3.7 Framework which is consistent across almost any device screen resolution.
